# SADL: Subject-Aware Distractor Localization

SADL is a benchmark for identifying visual distractors relative to a named subject. Given an image and a subject description *M*, the task is to localize which other objects distract from *M* while retaining compositionally essential ones.

- **1,000 images** · **1,800 subject-aware cases** · **14,617 annotated candidates**
- Annotation schema: 5 inclusion factors (F1–F5) and 3 exclusion rules (E1–E3)
- 7 VLMs evaluated zero-shot + 4 non-VLM baselines
